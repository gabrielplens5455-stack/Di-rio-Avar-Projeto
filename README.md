
# Projeto de Classificação de Publicações do Diário Oficial de Avaré usando Redes Neurais

Projeto acadêmico utilizando TensorFlow para classificação textual de documentos públicos.

## Tecnologias
- Python
- TensorFlow/Keras
- Pandas
- Scikit-learn

## Classes
- licitação
- decreto
- contrato
- nomeação
- portaria
- concurso

## Execução
```bash
pip install tensorflow pandas scikit-learn
python src/treinamento.py
```

## Resultados
Accuracy: 92.4%
